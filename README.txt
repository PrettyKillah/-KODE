Приложение обработки списка объектов
Выполнил Коновалов Даниил специально для KODE

--------Описание
Программа для чтения, группировки и сохранения объектов из файла.

-------Функционал
- Чтение объектов из "objects.txt"
- Добавление новых объектов
- 4 вида группировки:
  - По расстоянию (до 100, 1000, 10000, слишком далеко)
  - По имени (по первой букве, группа "#")
  - По времени (сегодня, вчера, неделя, месяц, год, ранее)
  - По типу (группа если >= 2 объектов, иначе "Разное")
- Сохранение результата в "result.txt"

## Запуск через MSYS2 
g++ main.cpp -o main
./main
